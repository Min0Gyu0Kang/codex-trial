Snake Game (Portable Windows Build)
===================================

How to run:
1. Unzip this folder.
2. Double-click launch_snake.bat

Controls:
- Move: Arrow Keys or WASD
- Pause/Resume: P (or Pause button)
- Restart: R (or Restart button)

Notes:
- Requires a modern browser installed on Windows.
- No internet connection required after unzipping.
